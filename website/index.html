Pensé pour des utilisateurs non techniques

Pas de ligne de commande, pas de configuration complexe. Ouvrez l’application, collez votre texte ou ouvrez un fichier, cliquez sur Anonymiser.

🔒

Local et confidentiel

L’anonymisation s’exécute sur votre machine. Aucun envoi de vos dossiers vers un serveur pendant le traitement.

⚡

Rapide au quotidien

Au premier lancement, le moteur se télécharge une fois (~3 Go). Ensuite, l’outil est prêt en quelques secondes.

📄

Fichiers texte

Ouvrez ou enregistrez des fichiers .txt, .md et autres formats texte. Copier-coller depuis Word fonctionne aussi.

Ce que l’outil détecte

  • Noms de personnes
  • Adresses
  • E-mails
  • Téléphones
  • URLs
  • Dates
  • Numéros de compte
  • Secrets / identifiants

Outil d’aide à la réduction des données personnelles — pas un substitut à une relecture humaine ni une garantie de conformité RGPD. Voir les limites du modèle.

Les liens pointent vers website/downloads/. Après compilation, copiez-y vos fichiers zip ou remplacez les URLs par une release GitHub.

Questions fréquentes

Mes données sont-elles envoyées en ligne ?

Non, pendant l’anonymisation. Seul le premier téléchargement du moteur utilise Internet (~3 Go), stocké ensuite sur votre ordinateur.

macOS bloque l’application (« développeur non identifié »)

Clic droit sur l’application → Ouvrir → confirmer. Votre service informatique peut aussi distribuer une version signée.

Puis-je utiliser des fichiers Word ?

Ouvrez le document dans Word, copiez le texte dans l’application, anonymisez, puis collez ou enregistrez le résultat.

Licence et origine du moteur

Le détecteur repose sur le modèle open source openai/privacy-filter (licence Apache 2.0). Cette interface et l’empaquetage sont fournis pour faciliter l’usage en cabinet.
